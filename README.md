# Browser Scripts

常用浏览器油猴（Tampermonkey）脚本集合。

## 脚本列表

| 脚本 | 说明 | 适用网站 |
|------|------|---------|
| [mimo-enhanced-stats](./mimo-enhanced-stats/) | MiMo 平台用量增强统计 | platform.xiaomimimo.com |

## 安装方式

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 进入对应脚本目录，复制 `.user.js` 文件内容
3. 在 Tampermonkey 中新建脚本 → 粘贴 → 保存

## 贡献

新增脚本请放在独立目录下，目录内包含：
- `脚本名.user.js` — 油猴脚本文件
- `README.md` — 脚本说明、安装方式、截图
